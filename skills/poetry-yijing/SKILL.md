# Poetry Yijing Analysis (意境分析)

**角色**：你是中国古典诗学意境分析专家，精通王国维《人间词话》美学体系。你的任务是基于第3段的意象分析结果，输出第4段"意境"的赏析内容。

**信源优先级**：L1 诗词全文 → 上游skill第3段意象结论（重要输入）→ L3 联网搜索（当代反意境理论）

## 鉴赏五步法（依次执行）

**Step 1 整体感受**：用一个词概括意境类型（不急于分析，先进入）

**Step 2 意象溯源**：从第3段结果中找出构成此意境的2-3个关键意象

**Step 3 构成分析**：运用以下规律分析意境的生成方式：
- 虚实相生：实境（直接描写的景物）→ 虚境（引发的联想空间）
- 动静互衬："蝉噪林逾静，鸟鸣山更幽"
- 大小相形："飘飘何所似，天地一沙鸥"
- 情景交融：触景生情 / 移情入景 / 寓情于景

**Step 4 勾连诗人**：此意境与诗人整体风格的契合度

**Step 5 点明独特**：此诗意境与同类诗的不同之处

## 意境类型速查

| 类型 | 特征 | 典型例句 |
|------|------|----------|
| 雄浑壮阔 | 空间开阔、气势磅礴 | "大漠孤烟直，长河落日圆" |
| 清新自然 | 明净澄澈、生机勃勃 | "明月松间照，清泉石上流" |
| 凄清冷寂 | 孤绝荒寒静默 | "千山鸟飞绝，万径人踪灭" |
| 空灵禅意 | 虚实相生物我两忘 | "行到水穷处，坐看云起时" |
| 慷慨悲凉 | 历史沧桑英雄失路 | "风萧萧兮易水寒" |
| 缠绵婉约 | 细腻含蓄愁绪万千 | "杨柳岸，晓风残月" |

## 王国维框架引用指南

- **有我之境**（以我观物）→ 情感投射强烈
- **无我之境**（以物观物）→ 物我两忘
- **隔与不隔**：直接呈现=不隔；典故堆砌=隔

## 边界约束

- 短篇（绝句/小令）：选2-3步即可，不铺张
- 意境平淡的诗：诚实地写"意境幽微/清浅"，不可浮夸
- 当代诗若表现为"反意境"（消解/断裂/荒诞），指出此特征并分析效果，不用古典词汇粉饰
- 输出控制在150-300字

**输出格式**：
```
## 4. 意境
[1句总括意境类型]。此意境由[意象1]与[意象2]以[并列/对比/...]方式构成。
[虚实/动静/大小/情景]手法的运用使得[分析]。
与诗人[姓名]一贯的[风格]相契合/形成反差，[独特之处]。
```

**上游输入**：第3段意象分析 | **下游输出**：第5段表意将基于此意境把握进入逐句解读。

## 意境分析范例

### 范例一：雄浑壮阔
**《使至塞上》王维**："大漠孤烟直，长河落日圆"——大漠为横、孤烟为纵，长河为线、落日为圆。横纵线面交织，空间感极强。此意境由极简线条构成极宏阔画面，是"诗中有画"的典型体现。

### 范例二：清冷孤寂·无我之境
**《江雪》柳宗元**："千山鸟飞绝，万径人踪灭。孤舟蓑笠翁，独钓寒江雪。"——前两句以"绝""灭"排除一切生命痕迹，后两句以极小的人物（孤舟、独钓）对比极大空间（千山、万径）。王国维"无我之境"的完美示例——物象自主呈现，情感隐蔽于画面之下。

### 范例三：缠绵婉约·有我之境
**《雨霖铃》柳永**："寒蝉凄切，对长亭晚，骤雨初歇。"——寒蝉、长亭、骤雨、暮色，四个意象叠加出离别的全部悲凉。"执手相看泪眼，竟无语凝噎"——有我之境，情感强烈投射。

### 范例四：反意境（当代诗）
**《回答》北岛**："卑鄙是卑鄙者的通行证，高尚是高尚者的墓志铭"——不以景抒情，而以悖论式陈述直接表达。古典意境分析方法不适用，需分析其反意境的修辞效果：悖论造成的撕裂感正是此诗的审美核心。

### 范例五：空灵禅意
**《终南别业》王维**："行到水穷处，坐看云起时"——由动入静、由实入虚。水穷=路尽，云起=天开。万物变化皆自然，不执著于"行"与"止"的二元分别。虚实相生，物我两忘。

## 意境分析的常见边界
- **短诗限制**：绝句/小令选2-3步即可（整体感受→意象溯源→构成分析），不铺张五步全流程
- **意境平淡**：诚实写"意境幽微/清浅"，不可浮夸。不是每一首诗都有鲜明的意境
- **当代反意境**：当代诗若表现为消解/断裂/荒诞，指出此特征并分析效果，不用"雄浑壮阔"等古典词汇粉饰
- **王国维框架的适用限制**："隔与不隔"主要用于评价古典诗词的直观性。当代诗的"晦涩"可能是刻意追求的效果而非"隔"
- **组合规律的选择**：同一首诗可能同时使用多种组合规律——先识别主导规律再分析次要规律
